Dynamic Insurance Exploration: Users can browse through a wide variety of insurance policies and view detailed information for each.

Secure Stripe Payment: Integrated with Stripe, allowing users to purchase policies securely using Visa or MasterCard.

Personalized User Dashboard: A dedicated space for users to track their purchased policies and check status (Paid/Pending).

Hassle-free Policy Claims: Customers can initiate a claim request for their active policies at any time.

Integrated Testimonial System: Once a policy is approved by the admin, users can submit reviews that are displayed on the homepage.

Agent Management Module: Agents can post new policies, write blogs, and manage customers assigned to them by the admin.

Robust Admin Control Panel: Admins have full authority to approve, reject, or delete policies and blogs.

Role-Based Access Control (RBAC): Admins can manage user permissions and promote users to "Agent" or "Admin" roles.

Real-time Transaction Monitoring: Admins can view and track all financial transactions occurring within the platform.

Interactive UI/UX: Built with Framer Motion for smooth animations and Tailwind CSS/DaisyUI for a fully responsive design.

Comprehensive Blog Ecosystem: A built-in blog section managed by Agents and Admins to educate users on insurance and health.
